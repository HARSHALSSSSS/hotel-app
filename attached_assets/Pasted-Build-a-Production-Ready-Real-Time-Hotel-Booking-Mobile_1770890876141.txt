🔗 API LAYER
Use GraphQL Gateway

All microservices connect to one GraphQL Gateway

Use Apollo Federation

Single endpoint for mobile app

Example:

POST /graphql


GraphQL should:

Combine data from multiple services

Handle batching

Resolve nested queries

🧠 BFF Pattern

Create a BFF layer specifically for mobile.

Responsibilities:

Transform GraphQL responses

Reduce over-fetching

Aggregate service calls

Handle device-specific optimizations

Flow:

Mobile App → BFF → GraphQL Gateway → Microservices
